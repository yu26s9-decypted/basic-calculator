# Basic Calculator

A command-line calculator built in Java that performs basic arithmetic operations using user input.

## About This Project

Created at **Year Up United — Learn to Code Academy (LTCA)** as an early Java exercise. This project introduced reading user input from the console and using control flow logic to branch between different operations.

## What I Learned

- Reading user input with `Scanner`
- Using a `switch` statement to handle multiple operation choices
- Working with `double` for decimal arithmetic
- Building interactive, menu-driven console applications
- Understanding program flow with `while` loops and `return` statements

## How to Run

```bash
mvn compile exec:java -Dexec.mainClass="BasicCalculator"
```

Or open in IntelliJ IDEA and run `BasicCalculator.java`.

## Features

- Accepts two numbers from the user
- Supports four operations: **Addition**, **Subtraction**, **Multiplication**, **Division**
- Menu-driven interface with single-character input (`A`, `S`, `M`, `D`)

## Tech Stack

- **Java**
- **Maven**
- **IntelliJ IDEA**
